Pokédex App

The Pokédex App is an interactive JavaScript application that lets users explore a wide range of Pokémon. Using the PokéAPI, the Pokédex App dynamically loads Pokémon data in form of a list.

Project Objective

The goal of the Pokédex App is to make Pokémon exploration easy and engaging while demonstrating key JavaScript functionalities. The app allows users to:

  • Browse a dynamically loaded list of Pokémon
  • View Pokémon details (name, height, and image) in a Bootstrap-powered modal
  • Search for specific Pokémon by name
  • Swipe through Pokémon within the modal
  • Enjoy a responsive design with a smooth navigation bar and a music button for the perfect feeling

Technology Used

To ensure a smooth and efficient user experience, the Pokédex App is built using:

  • JavaScript (ES6) - Handles API requests, DOM manipulation, and event handling
  • HTML5 & CSS3 - Provides structure and styling
  • Bootstrap 5.3 - Ensures a responsive layout and interactive modals
  • PokéAPI - Fetches real-time Pokémon data
  • GitHub Pages - Hosts the application
