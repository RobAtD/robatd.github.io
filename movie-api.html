movie API

The movie API is a backend service built with Node.js, Express, and MongoDB, designed to handle user authentication and movie data for the myFlix application. The API enables users to register an account, log in, browse movies, and manage their favorite films.

Project Objective

The movie API's goal is to provide a structured and scalable backend, that supports:

  • CRUD operations for managing movies, users and favorite movies
  • Secure user authentication and authorization with JWT and Passport.js
  • Database management using MongoDB and Mongoose
  • RESTful endpoints that integrate seamlessly with the myFlix Client
  • Cross-origin support for external clients via CORS

Technology Used

  • Node.js & Express - For handling API requests
  • MongoDB & Mongoose - For storing and managing movie/user data
  • JWT & Passport.js - For secure authentication
  • CORS - To enable cross-origin API requests
  • MongoDB Atlas - Cloud-based database storage
  • Heroku - For backend deployment
