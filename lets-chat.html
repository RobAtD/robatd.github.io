Let's Chat!

Let's Chat! a mobile chat application built using React Native, Expo, and Firebase. It allows users to send text messages, images and share their location in real-time, with offline support for message storage.

Project Objective

The goal of Let's Chat! is to provide a seamless chat experience with the following features:

  • Anonymous login
  • Real-time messaging
  • Offline message storage
  • Sending images and location sharing

Technology Used

  • React Native - A framework for cross-plattform mobile development
  • Expo - A platform for making universal native apps
  • Firebase Firestor - A Real-time cloud database
  • Gifted Chat - A library for chat UI components
  • AsyncStorage - Local storage for offline functionality

Reflections on the project

The development of Let's Chat! was my first own project in the direction of app development. I had already worked on a mobile app with a friend before my studies, but back then I wasn't as familiar with the subject as I am now. The introduction to React Native was very easy for me, as there were many similarities to React. The biggest challenges were that when simulating the application in the browser, errors were often not visible at first glance and only became apparent when simulating in Android Studio. Initially, I always simulated the application on the web, as Android Studio consumed more of my laptop's resources, but later on, simulating directly via Android Studio proved to be a better option, as it allowed the causes of errors to be found more quickly.
